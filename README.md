> 👨‍💻 Como rodar o projeto 
  
* Clonar o repositório

* Rodar `npm install`

* Criar um arquivo `.env` na pasta raiz do projeto:

 ``` 
   PORT = 3003
   DB_HOST = 
   DB_USER = 
   DB_PASSWORD = 
   DB_DATABASE = 
   JWT_KEY = "secret"
```

* Criar tabelas no arquivo ```tables.sql```

* Rodar `npm start` para ligar o servidor
